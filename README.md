# minha-calculadora
Projeto da calculadora do Curso Analista de Dados da EBAC

## Execução do arquivo "executar_calculadora.sh":
  Primeiro dê permissão de execução do arquivo ao seu usuário:
    `chmod +x executar_calculadora.sh`
  No terminal use o comando `./executar_calculadora.sh`
  O programa vai iniciar solicitando a entrada de dados

## Arquivo `calculadora.py`
  Este script calcula as operações aritiméticas básicas de adição, subtração, multiplicação e divisão.
  Foi criado  com a lingyagem Python.
